# Screenshots

This folder contains screenshots and demo images for the Argo Access Control - Approval Handler project.

## How to Add Screenshots

1. Take screenshots of:
   - Slack slash command modal
   - Access request in Slack channel
   - Approval buttons interface
   - Admin dashboard
   - Request history page
   - Rule management UI
   - Argo Workflow execution

2. Save images with descriptive names:
   - `slack-request-modal.png`
   - `slack-approval-flow.png`
   - `admin-dashboard.png`
   - `rules-management.png`
   - `request-history.png`
   - `argo-workflow-running.png`

3. Reference them in the main README.md:
   ```markdown
   ![Slack Request Modal](screenshots/slack-request-modal.png)
   ![Admin Dashboard](screenshots/admin-dashboard.png)
   ```

## Recommended Image Specs

- **Format**: PNG (for UI screenshots) or JPG (for photos)
- **Size**: Max 1920x1080px (resize larger images)
- **File size**: Keep under 500KB per image (use compression)
- **Tools**: Use [TinyPNG](https://tinypng.com/) to compress images

## Demo GIF

For animated demos:
- Use [LICEcap](https://www.cockos.com/licecap/) (Mac/Windows)
- Or [Kap](https://getkap.co/) (Mac)
- Keep GIFs under 5MB

Example:
```markdown
![Request Flow Demo](screenshots/demo-flow.gif)
```
